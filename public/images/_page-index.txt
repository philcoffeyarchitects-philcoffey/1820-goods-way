HOW TO PLACE IMAGES (drag & drop)
  1. Run the deck locally:   npm run dev   (http://localhost:4321)
  2. Navigate to a page and DRAG an image file from your computer straight
     onto the placeholder. It saves automatically into this folder + manifest.
  3. Hover the image to choose "Fit all" (whole image shown) or "Fill"
     (fills the frame, crops overflow), or ✕ to remove it.
  4. Commit the changes (public/images/*) to GitHub — the images then show
     for everyone, on any clone or deploy. (Drag-drop editing only works in
     local `npm run dev`; the deployed site is view-only.)

  No file-naming needed — dropping onto a placeholder targets that exact slot.
  Pages marked "2 / 3 / 5" below have multiple slots; drop onto each one.
  Formats: .jpg / .png / .webp / .gif / .svg. Large images are auto-downscaled.

"text" pages have no image — skip them.
